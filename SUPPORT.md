# Support

## Getting Help

There are several ways to get help with this project:

### Documentation

- **README.md** - Getting started and basic usage
- **CONTRIBUTING.md** - How to contribute to the project
- **docs/** - Additional documentation and guides

### Community Support

- **GitHub Discussions** - Ask questions, share ideas, and connect with other users
- **GitHub Issues** - Report bugs or request features

### Reporting Issues

Before creating a new issue:

1. Search existing issues to avoid duplicates
2. Use the appropriate issue template
3. Provide as much detail as possible

### Feature Requests

We welcome feature suggestions! Please:

1. Check if the feature has already been requested
2. Clearly describe the use case
3. Be open to discussion about implementation

## Response Times

This is an open source project maintained by volunteers. While we strive to respond quickly:

- **Bug reports**: Usually within 1 week
- **Feature requests**: Reviewed periodically
- **Security issues**: Within 48 hours (see SECURITY.md)

## Commercial Support

For commercial support or custom development, please reach out via GitHub Discussions.

## Stay Updated

- Watch the repository for updates
- Check the CHANGELOG for new releases
- Follow discussions for announcements
